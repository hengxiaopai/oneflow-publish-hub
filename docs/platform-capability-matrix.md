更新日期：2026-06-13

## 判定口径

- `是`：已有明确的产品内接入方案，或存在可核验的官方能力。
- `半自动`：官方存在相关能力，但仍依赖应用审核、账号授权、用户确认、平台审核或人工补充。
- `待验证`：未找到足够的官方公开资料，或尚未使用真实账号完成端到端验证。
- `生成草稿`：默认指 OneFlow 本地生成平台版本，不等同于已写入第三方平台草稿箱。
- 本矩阵是 MVP 的保守产品策略，不是对第三方平台长期能力的承诺。平台规则和开放范围可能变化，接入前必须再次核验。

## 产品决策

1. MVP 仅将自建 Blog 标记为已验证自动发布。
2. 第三方文章平台默认走“本地生成草稿 → 人工确认 → 复制或平台侧提交”。
3. 抖音与哔哩哔哩即使具备官方发布能力，也归类为半自动，因为授权、审核和用户确认不能被省略。
4. 小红书通用笔记发布能力未完成官方资料与真实账号验证，因此保持复制发布。
5. 每个第三方渠道上线前，需要记录真实账号、权限申请、接口范围、审核要求、失败码和回滚方式。
